lolcloud
========

An astonishgly simple open-source spinoff of Dropbox.
Secure storage for pennies.
Sync directories as you damn-well please, sir.

## Installation

1. Clone it
    $ git clone https://github.com/sweatshirtio/lolcloud && cd lolcloud
2. Install it
    $ rake install

## Usage

1. Setup storage
    $ todo
2. Execute
    $ lolcloud
3. Be amazed?
